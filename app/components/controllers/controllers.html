Components The Meat of your application

You have now seen that when you bind an element to a property on a component using either ng-model="component.someProp" or with the double curlies {{}} your page will automatically update

Lets now dive into a component and start writting some functions

  • controller.js

    
				(function(){
					angular.module('angularPlayground')
						.component('controllersComponent', {
							controller: ControllersController,
							templateUrl: "app/components/controllers/controllers.html"
						});

						function ControllersController(){
							//Write your functions here ... 
							//Remember this function is a constructor.
							//Because of the way angular was written, 
							//only the properties put in this constructor will be accessible to the view!
							}
						}());
